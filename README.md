# 🧠 Brain Tumor Classification System

This is a Streamlit-based web app for brain tumor classification using MRI images.

## 🚀 Features
- Upload MRI image (.jpg, .png, .mat)
- Automatic feature extraction
- ML model prediction
- Confidence score

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
